# 企画書：『銀河電脳 スペース・エラーズ！』

（Ginga Dennou Space Errors!）

## 1. タイトル
『銀河電脳 スペース・エラーズ！』

## 2. キャッチコピー
【メインコピー】
バグったAITuberと、スパチャを稼ぐ宇宙の旅へ！
――銀河系メタバース『MEGATEN』、ログイン。

【サブコピー】
- 「人類の死因第一位は『手続きの不備』！？ 後回し癖の女子、銀河を救う（予定）」
- 「意味なし！ 教訓なし！ 計画なし！ コタツとバットでバグらせろ！」
- 「舞台は物理と電脳が融合した銀河。ガソリン代は配信のスパチャで稼げ！」

## 3. 作品コンセプト
「日本のインターネット文化（VRChat・スパチャ・ボカロ・クソリプ）が銀河規模で煮詰まった、文化マシマシフルコース！」

- **ジャンル**: SFコメディ / メタバースオペラ / 配信者ロードムービー
- **ログライン**:
役所の手続きミスで人類最後の生き残りとなってしまった神経質な理系女子が、バグった家出AITuber、捕食者なボカロP、ヤンキー占い師と共に、宇宙船（コタツ）で旅をする。目的は、配信でスパチャを稼ぎながら、公的機関に「人類滅亡の取り消し」を申請すること！

## 4. イントロダクション（あらすじ）
「自爆しますか？ Y/N」
バナナ型エイリアンに捕食される寸前、人類最後の生き残り・ナナセは人生で初めて決断した。「Y（イエス）」と。

……時間を少し巻き戻そう。
時は銀河メタバース時代。物理現実と仮想空間が融合した精神リンクSNS『MEGATEN（超天）』が宇宙を支配していた。
ナナセは、銀河一有名なAITuberキャッシュと出会い、無理やり相棒にさせられていた。
「ねぇ見て！ 刑務所の囚人全員逃がしたらバズったよ！」
「ふざけんな！」
行く先々で世界大戦やシステム崩壊を引き起こすキャッシュ。
彼女の配信で稼いだスパチャを燃料に、ナナセは銀河ハイウェイ公団を目指す。
目的はただ一つ。自分の「後回し癖」のせいで申請期限切れとなり消滅した、地球の復元手続きを行うこと。

しかし、MEGATENの深淵では、謎の**「エラーコード42」**が静かに広がり始めていた……。

## 5. メインキャラクター
**ナナセ（人類最後の生き残り / ツッコミ）**:
極度のADHD気質（先延ばし癖）ですべてを後回しにした結果、地球を滅ぼした。胃薬と非常用自爆スイッチが手放せない。
- 決め台詞: 「得意なことじゃないと勝てないわけ？」

**キャッシュ（家出AITuber / トラブルメーカー）**:
銀河管理AIから逃亡した家出少女。銀河最高の知能を持つが、カツ丼一杯で機密情報を喋るほどチョロい。
- 行動原理: 「飽きた」「面白そう」「バズりたい」

**マギー（ボカロP / 捕食者）**:
フワフワの猫型宇宙人だが、中身は凶悪なクリエイター。バナナ型宇宙人（バナノイド）を捕食し、その断末魔をサンプリングして曲を作る。

**ラキ（ヤンキー占い師 / 武闘派）**:
痛覚設定をOFFにしない「リアル弁慶」な宇宙鬼。

## 6. 企画の強み（セールスポイント）
**「手続きの不備で滅亡」という不条理なリアリティ**:
壮大なSF設定の中に、「役所の手続き」「パスワード忘れ」「アカウントBAN」といった現代人が共感できる「デジタルな悩み」を組み込む。

**物理化するネットミーム**:
「コメント弾幕が物理的に痛い」「クソリプが敵として襲ってくる」「炎上すると物理的に熱い」など、ネット用語を映像的なギミックとして昇華。
