# 🌌 銀河電脳 スペース・エラーズ！ (Space Errors!)

**「意味なし！ 教訓なし！ 計画なし！ コタツとバットでバグらせろ！」**

## 📖 作品概要
人類最後の生き残りとなってしまった神経質な理系女子が、バグった家出AITuber、捕食者なボカロP、ヤンキー占い師と共に、宇宙船（コタツ）で銀河系メタバース『MEGATEN』を旅するSFコメディ。

* **ジャンル:** SFコメディ / メタバースオペラ / 日常系
* **ターゲット:** 日本のネット文化とSFを愛するすべての人類
* **現在のステータス:** 執筆中 ✍️
