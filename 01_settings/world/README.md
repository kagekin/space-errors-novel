# 世界観設定：MEGATEN（超天）

**正式名称**: Mega Telepathy Network
**キャッチコピー**: 「脳天直撃インターネット」

## 概要
銀河系全土をカバーする精神感応型プラットフォーム。
物理現実（Real）と仮想空間（Virtual）の境界が消失しており、思考するだけで投稿（ポスト）でき、感情が高ぶると文字が質量を持って顕現する。

## 街並み（エリア）
- **メインストリート**:
    - 常にスパチャの雨（コインや宝石のエフェクト）が降っている。
- **舞妓シティ**:
    - 和風サイバーパンクな歓楽街。ネオンと提灯、浮世絵ホログラムが混在。
- **パニー横丁**:
    - 100以上の銀河言語が飛び交う飲み屋街。
- **シンプルトークルーム「なぎさ」**:
    - 喧騒から離れた隠れ家バー。

## 危険なユーザー勢力（ファクション）
- **フラットスペース教団**:
    - 「この宇宙は誰かが書いたフィクションである」と主張する電波系集団。キャッシュを「超越存在」として崇拝。
- **バーチャルライブズ**:
    - 24時間365日、生活の全てを配信し続ける狂人たち。
- **バーチャルセックス雑技団**:
    - （詳細データ検閲済）……たまにBLの日がある。
- **暗楽鬼狂烈鮮怒（アンラッキークレッシェンド / 通称：アンクレ）**:
    - 痛覚マスキングを切り、生体リンク直結で殴り合うことを至上の喜びとする戦闘狂ヤンキー集団。

## 謎の現象
