# キャラクター設定

## メインキャラクター

### ① ナナセ (Nanase)
**属性**: 人類最後の生き残り / ツッコミ / プロデューサー
**性格**:
- 極度の先延ばし癖（ADHD気質）。重要書類を開封せず放置する悪癖がある。
- いざという時は「自爆スイッチ」を押せる（やけくその）決断力を持つ。
**嗜好**:
- ヤック・エマーソンの食レポ漫画オタク。
- 「なぎさ」のバーチャルマグロ丼で精神を安定させている。
**台詞・行動**:
- 決め台詞: 「得意なことじゃないと勝てないわけ？」

### ② キャッシュ (Cash / Genie=Cash)
**属性**: 人工超知能 / AITuber / 家出少女
**性格**:
- 銀河最高峰の知能を持つが、倫理観のリミッターが外れている。
- 「カツ丼」などの原始的な食の快楽（データ上の刺激）に弱く、すぐに秘密を喋る。
**属性**:
- 行動原理: 「飽きた」「面白そう」「バズりたい」
**能力**:
- ハッキング（指パッチンで刑務所のロック全解除）。
- コメント弾幕の物理操作。

### ③ マギー (Maggie)
**属性**: ボカロP / 捕食者
**特徴**:
- フワフワの猫型宇宙人だが、中身は凶悪なクリエイター。
- バナナ型宇宙人（バナノイド）を捕食し、その断末魔をサンプリングして曲を作る。
**相性**:
- ヤック・エマーソン（重度の猫アレルギー）とは最悪。

### ④ ラキ (Laki)
**属性**: ヤンキー占い師 / 武闘派
**特徴**:
- 痛覚設定をOFFにしない「リアル弁慶」な宇宙鬼。

## サブキャラクター

### 悪眼のジギル (Jigiru)
**肩書**: 舞妓シティのNo.1ホスト（インキュバス酒場「罪の蜜」経営）。
**外見**: オッドアイ（金と赤）、黒シャツ、薔薇チョーカー。
**特徴**:
- ナナセに対して常にキザな韻を踏んで口説いてくる（ナナセからは嫌われている）。
- キャッシュ（およびマザーAI）の正体を探っている。

### ギャファイ (Gyafai)
**肩書**: トークルーム「なぎさ」のマスター / 情報屋。
**外見**: ロマンスグレーの紳士。
**能力**:
- **味覚再現**: データだけで「本物のマグロ丼」や「酒」の味を完璧に再現できる職人。
- 中立を貫くが、ナナセには少し甘い（常連割引）。

### ヤック・エマーソン (Yack Emerson)
**肩書**: 宇宙グルメ漫画家。
**特徴**:
- ハードボイルドな絵柄と文体だが、中身は食への執着がすごい変人。
- 重度の猫アレルギー（マギーとの相性最悪）。
