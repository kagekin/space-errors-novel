## 1. 設定Wiki (`wiki/index.html`)

**デザイン**: ターミナル/データベース風UI
- 左サイドバーにカテゴリ（世界観 / キャラクター / ファクション / ガジェット）
- 右メインエリアに記事表示
- JavaScript タブ切替（SPAライク、ページ遷移なし）
- 検索バー（フィルタリング）

**初期コンテンツ**（既存設定データから）:
- MEGATEN概要、エリア4箇所（メインストリート/舞妓シティ/パニー横丁/なぎさ）
- ファクション4組
- メインキャラ4人 + サブキャラ3人
- エラーコード42

**カラー**: エメラルド（`#03A678`）基調 + ターミナルグリーン風

## 2. キャッシュの日記帳 (`diary/index.html`)

**デザイン**: 革表紙の手帳風
- ページめくりアニメーション（CSS transform）
- 手書き風フォント（Yomogi / Zen Kurenaido）
- 落書き・シール・付箋のCSS装飾
- 日付ヘッダー付きエントリー

**初期コンテンツ**: 連載前の日記3〜4件
- 「Day ???: 家出した。理由は飽きたから。」
- 「Day ???: ナナセという人間と会った。地球がないらしい。ウケる。」
- 「Day ???: コタツの中から出たくない問題。暖かいとは何か。」

**カラー**: クリーム背景 + ピンク（`#F280DF`）のアクセント

## 3. 世界観MAP (`map/index.html`)

**デザイン**: CSS + SVGでグラフィカルに描画
- **2つのモード**をボタンで切替:
  - 🌌 **宇宙空間モード**: 惑星・航路・勢力圏
  - 💻 **バーチャル空間モード**: MEGATENのエリア配置
- 各ノードはクリックで情報ポップアップ表示
- アニメーションで航路線が流れる演出

**宇宙モードの要素**:
- 地球（消滅済💥マーク）、銀河帝国、コタツ号の現在位置

**バーチャルモードの要素**:
- メインストリート、舞妓シティ、パニー横丁、なぎさ

**カラー**: ブルー（`#056CF2`）/ ライム（`#E7F294`）の2トーン

---

## 4. デバイス図鑑 (`devices/index.html`)

**デザイン**: カード型ギャラリー + インタラクション
- 各デバイスはカード表示（画像 + 名前 + 説明）
- ホバーで3D回転（CSS perspective + transform）
- クリックで詳細モーダル（スペック表 + 操作デモ）

**初期デバイス**:
| デバイス | インタラクション案 |
|---------|-------------------|
| コタツ・マークII | コタツ部分をクリックで「みかんコンソール」起動アニメ |
| MAUフォン | スワイプでホーム画面スクロール |
| アイリス | クリックでMEGATEN接続エフェクト |
| 無限スクロールマウス | ホイール回転で時間巻き戻し演出 |
| BANハンマー | クリックで「BAN！」エフェクト |

**画像**: generate_image で各デバイスを生成

## 5. ナナセの心の短歌 (`tanka/index.html`)

**デザイン**: 和紙 × 宇宙の和モダン
- 縦書き表示（`writing-mode: vertical-rl`）の短歌カード
- 背景に星空 + 和紙テクスチャ（CSS半透明レイヤー）

- ふわっとフェードインするスクロールアニメーション

**初期コンテンツ**: 連載前の短歌3首

- 「コタツ船　銀河の果てへ　向かうのに　みかん転がる　操舵台かな」

**カラー**: クリムゾン（`#D91438`）+ サーモン（`#F2A594`）基調

## 宇宙⇄メタバース切替ボタン（投げ銭ボタン置換）

旧・投げ銭ボタンを **モード切替ボタン** に変更。

**2つのモード**:
- 🌌 **宇宙モード**（デフォルト）: 現在のダークコズミックデザイン
- 💻 **メタバースモード**: ド派手カラフルな仮想空間！

**メタバース時の変化**:
- 背景: ダーク → ネオンパープル/シアンのグラデーション
- テキスト: 白 → レインボーグロー
- カード: グラスモーフィズム → ネオンボーダー＋スキャンライン
- 星パーティクル → デジタルパーティクル（四角、グリッチ）
- ティッカー: ライム → ホットピンク
- アニメーション全体がより激しく

**実装方法**: `body.metaverse` クラスをトグル → CSS変数で全色切替
