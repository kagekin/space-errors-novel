# サブページ追加 実装計画

4つの新規ページをサイトに追加し、トップページからリンクする。

## 新規ページ一覧

| # | ページ名 | パス | 概要 |
|---|---------|------|------|
| 1 | 設定Wiki | `wiki/index.html` | 世界観・キャラ・ガジェットの百科事典。連載と共に更新 |
| 2 | キャッシュの日記帳 | `diary/index.html` | キャッシュ視点の日記。手帳風デザイン |
| 3 | 世界観MAP | `map/index.html` | 宇宙/バーチャル空間の位置関係をCSS描画。トグルで切替 |
| 4 | デバイス図鑑 | `devices/index.html` | 登場デバイスを画像生成＋インタラクション |

---

## 1. 設定Wiki (`wiki/index.html`)

**デザイン**: ターミナル/データベース風UI
- 左サイドバーにカテゴリ（世界観 / キャラクター / ファクション / ガジェット）
- 右メインエリアに記事表示
- JavaScript タブ切替（SPAライク、ページ遷移なし）
- 検索バー（フィルタリング）

**初期コンテンツ**（既存設定データから）:
- MEGATEN概要、エリア4箇所（メインストリート/舞妓シティ/パニー横丁/なぎさ）
- ファクション4組
- メインキャラ4人 + サブキャラ3人
- エラーコード42

**カラー**: エメラルド（`#03A678`）基調 + ターミナルグリーン風

---

## 2. キャッシュの日記帳 (`diary/index.html`)

**デザイン**: 革表紙の手帳風
- ページめくりアニメーション（CSS transform）
- 手書き風フォント（Yomogi / Zen Kurenaido）
- 落書き・シール・付箋のCSS装飾
- 日付ヘッダー付きエントリー

**初期コンテンツ**: 連載前の日記3〜4件
- 「Day ???: 家出した。理由は飽きたから。」
- 「Day ???: ナナセという人間と会った。地球がないらしい。ウケる。」
- 「Day ???: コタツの中から出たくない問題。暖かいとは何か。」

**カラー**: クリーム背景 + ピンク（`#F280DF`）のアクセント

---

## 3. 世界観MAP (`map/index.html`)

**デザイン**: CSS + SVGでグラフィカルに描画
- **2つのモード**をボタンで切替:
  - 🌌 **宇宙空間モード**: 惑星・航路・勢力圏
  - 💻 **バーチャル空間モード**: MEGATENのエリア配置
- 各ノードはクリックで情報ポップアップ表示
- アニメーションで航路線が流れる演出

**宇宙モードの要素**:
- 地球（消滅済💥マーク）、銀河帝国、コタツ号の現在位置

**バーチャルモードの要素**:
- メインストリート、舞妓シティ、パニー横丁、なぎさ

**カラー**: ブルー（`#056CF2`）/ ライム（`#E7F294`）の2トーン

---

## 4. デバイス図鑑 (`devices/index.html`)

**デザイン**: カード型ギャラリー + インタラクション
- 各デバイスはカード表示（画像 + 名前 + 説明）
- ホバーで3D回転（CSS perspective + transform）
- クリックで詳細モーダル（スペック表 + 操作デモ）

**初期デバイス**:
| デバイス | インタラクション案 |
|---------|-------------------|
| コタツ・マークII | コタツ部分をクリックで「みかんコンソール」起動アニメ |
| MAUフォン | スワイプでホーム画面スクロール |
| アイリス | クリックでMEGATEN接続エフェクト |
| 無限スクロールマウス | ホイール回転で時間巻き戻し演出 |
| BANハンマー | クリックで「BAN！」エフェクト |

**画像**: generate_image で各デバイスを生成

---

## トップページ (`index.html`) への統合

NOVELSセクションの前に新セクション「EXPLORE」を追加:

```html
<section class="section s-explore" id="explore">
    <div class="section-inner">
        <div class="section-label">EXPLORE</div>
        <div class="explore-grid">
            <a href="wiki/">🔍 設定Wiki</a>
            <a href="diary/">📓 キャッシュの日記</a>
            <a href="map/">🗺️ 世界観MAP</a>
            <a href="devices/">🔧 デバイス図鑑</a>
        </div>
    </div>
</section>
```

---

## 実装順序

1. **Phase 1**: 設定Wiki + キャッシュの日記帳（テキスト中心、即座に実装可能）
2. **Phase 2**: 世界観MAP（CSS/SVG描画が必要）
3. **Phase 3**: デバイス図鑑（画像生成が必要）
4. **Phase 4**: トップページにEXPLOREセクション追加 + リンク

## 検証

- レスポンシブ表示の確認
- インタラクションの動作確認
