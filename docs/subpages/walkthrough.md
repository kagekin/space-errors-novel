# Walkthrough - New Subpages & Metaverse Toggle

新しいコンテンツページとメタバースモードの実装が完了しました。

## 変更内容

### 1. メタバースモード (Phase 0)
- **トグル**: 右下の「宇宙モード」ボタンで切り替え可能
- **ビジュアル**:
  - 背景がデジタルグリッドに変更
  - ニコニコ動画風の流れるコメント
  - カードやウィンドウにネオンボーダーとスキャンライン追加
  - 星パーティクルがデジタルノイズに変化

### 3. トップページ統合 (Phase 4)
- **EXPLOREセクション**: キャラクタ紹介の上に新設。各サブページへのリンクを配置。
- **OGP**: `se_ogp.png` を設定済み。

## 検証手順

1. **トップページ**:
   - 右下の「宇宙モード」ボタンをクリックし、画面が劇的に変化することを確認（流れる文字、背景グリッドなど）。
   - 「EXPLORE」セクションの各リンクをクリックし、ページ遷移を確認。

2. **サブページ**:
   - 各ページのデザインテーマが異なることを確認。
   - 「TOPへ戻る」ボタンで戻れることを確認。
   - **Wiki**: メニュークリックで内容が切り替わるか。
   - **Map**: ピンをクリックしてモーダルが出るか。
   - **Yankee**: 「挑戦状」ボタンでアラートが出るか。
