# 実装計画書: 『銀河電脳 スペース・エラーズ！』 特設ティザーサイト

「銀河特急ミルキーサブウェイ」のようなレトロかつフューチャーな雰囲気で、メタバース×宇宙×コメディを感じさせるティザーサイトを構築します。

## ユーザーレビューが必要な事項
- デザインの方向性（「レトロフューチャー」「メタバース」「宇宙」のバランス）

## 変更内容

### プロジェクトルート
#### [MODIFY] index.html
ベンチマークサイト（銀河特急ミルキーサブウェイ、超かぐや姫）を意識し、**「没入感」「カオス」「エンタメ」** を最重視したデザインへ刷新します。

**デザインコンセプト定義**:
- **テーマ**: 『銀河の裏路地にあるラクガキだらけの電脳壁』
    - 「MEGATEN」のストリートエリア（パニー横丁）に貼られたポスターのようなサイト。
    - **Splatoon** のようなインク/ペンキの質感 × **銀河系漫遊指南** の不条理なユーモア。
- **キーワード**:
    - **Galactic Street**: ヒップホップ、グラフィティ、ステッカーボム。
    - **Pop & Chaos**: 原色バリバリ、斜めに配置、重力無視。
    - **Don't Panic**: エラーが起きても「まあいいか」で済ませる脱力感。

**詳細デザイン仕様**:

1.  **カラーパレット**:
    -   **ベース**: Concrete Grey/Dark Asphalt - コンクリートやアスファルトの質感。
    -   **インク1**: Hot Pink (`#ff00ff`) - 飛び散るインク。
    -   **インク2**: Cyan Blue (`#00ffff`) - 飛び散るインク。
    -   **インク3**: Danger Yellow (`#ffcc00`) - 警告テープ。

2.  **タイポグラフィ**:
    -   **見出し**: 極太のゴシック体（インパクト重視）を斜めに配置。
    -   **本文**: 手書き風フォントや、マーカーで書いたような文字。

3.  **UI/レイアウト構造**:
    -   **グリッド破壊**: 整列させない。要素をステッカーのようにペタペタ貼った感じ。
    -   **背景**:
        -   宇宙空間だが、手前に金網（フェンス）やスプレーアートがある。
        -   スクロールするとインクが垂れてくる。

4.  **インタラクション**:
    -   **マウスストーカー**: スプレー缶の照準になり、クリックするとインクが出る（擬似落書き）。


5.  **セクション構成案**:
    -   **Hero**: 巨大なロゴ + キャッチコピー「バグったAITuberと...」 + 背景動画/Canvas。
    -   **News (Ticker)**: 電光掲示板風に流れる更新情報。
    -   **Novel (Main)**:
        -   「連載」エリア。最新話へのリンクをカードデザインで。
        -   外部サイト（カクヨム、TALES）へのリンクボタンを目立たせる。
    -   **Characters**:
        -   主要4名を「キャラ選択画面」風に並べる。選択（ホバー）で詳細表示。
    -   **Media Mix (Coming Soon)**:
        -   「Access Denied / Coming Soon」のテープが貼られた立ち入り禁止エリア風。
    -   **Footer**:
        -   SNSシェアボタン（大きく、押しやすく）。
        -   コピーライト。
