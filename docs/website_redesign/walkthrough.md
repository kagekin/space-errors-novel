# 検証記録: 特設サイト リデザイン

「銀河電脳 スペース・エラーズ！」特設サイトのリデザインを行いました。
銀河特急ミルキーサブウェイを参考に、レトロフューチャーかつメタバース的なデザインを実装しました。

## 変更内容

### 新規作成ファイル
1.  **[index.html](file:///d:/github/space-errors-novel/index.html)**
    *   トップページ。グリッチエフェクト、ニュースティッカー、限定されたキャラクター紹介（主要4名のみ）を実装。
2.  **[style.css](file:///d:/github/space-errors-novel/style.css)**
    *   ネオンカラー、スキャンライン、ドットフォントを使用したスタイル定義。
3.  **[script.js](file:///d:/github/space-errors-novel/script.js)**
    *   Canvasによる星空背景描画。
    *   `updates.json` の非同期読み込みとニュースティッカーへの反映。

### デザインのポイント
*   **レトロ感**: ドットフォント (DotGothic16)、スキャンラインオーバーレイ。
*   **宇宙・サイバー**: 星空背景、ネオンカラー、グリッチテキスト。
*   **キャラクター**: 主要4名（ナナセ、キャッシュ、マギー、ラキ）のみを表示。
*   **動的要素**: ニュースティッカーが `updates.json` の内容を流し続けます。

## 検証手順

1.  ブラウザで `index.html` を開いてください。
2.  **星空背景** が描画され、奥から手前に星が流れていることを確認。
3.  `SPACE ERRORS!` のタイトルが時折 **グリッチ（ノイズ）** することを確認。
4.  画面中央の **ニュースティッカー** に `updates.json` の内容（第1話公開予定など）が流れていることを確認。
5.  **キャラクター紹介** がナナセ、キャッシュ、マギー、ラキの4名のみであることを確認。

## 次のステップ

*   コンテンツ（連載小説）の追加に合わせて `updates.json` を更新してください。
*   `public_website` フォルダはバックアップとして残っていますが、ルートでの運用が安定したら削除を検討してください。
