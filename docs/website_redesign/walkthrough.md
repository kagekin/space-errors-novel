# 検証記録: 特設サイト リデザイン（レトロポップ版）

## 概要
「銀河電脳 スペース・エラーズ！」特設サイトを **レトロポップ** テーマで実装しました。

## デザイン変遷
1. **v1**: ダーク × ネオン（サイバーパンク） → ダサいと却下
2. **v2**: コンクリート × インク（ストリート） → まだ暗すぎると却下
3. **v3（現行）**: クリーム × 暖色（レトロポップ） ✅ 採用

## 実装ファイル

| ファイル | 内容 |
|---------|------|
| [index.html](file:///d:/github/space-errors-novel/index.html) | 全セクション構成（Hero / 世界観 / 連載 / キャラ / メディア） |
| [style.css](file:///d:/github/space-errors-novel/style.css) | レトロポップCSS（Reggae One + Zen Maru Gothic） |
| [script.js](file:///d:/github/space-errors-novel/script.js) | インタラクション（インク飛沫、スパチャ、シェア） |

## 主な機能
- 🎨 **クリック＝インク飛沫**: 8色のカラフルなスプラッシュ
- 💰 **擬似スパチャ**: キャラセリフがランダム表示
- 📱 **SNSシェア**: X / LINE / リンクコピー
- 🌌 **世界観セクション**: MEGATENの設定紹介 + キーワードタグ
- 🔒 **Media Mix**: Coming Soonバッジのバウンスアニメーション
