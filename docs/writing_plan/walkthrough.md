# 執筆計画ファイルの使い方

ご要望の「執筆計画を立てる場所」として、以下のファイルとフォルダを用意しました。

## 1. 全体計画・進捗管理 (`00_planning/trilogy_plan.md`)
このファイルがメインの管理シートになります。

- **作品概要**: タイトルやジャンル、目標文字数などを記載しています。
- **各巻の管理表**: 
  - 章ごとのタイトル、プロット、予定文字数、進捗状況を一覧で管理できます。
  - 執筆が進んだら `[ ]` にチェックを入れたり、文字数を更新してください。

## 2. 原稿の保存場所 (`02_manuscripts/`)
各巻ごとにフォルダを分けました。執筆時はここを使ってください。

- `volume_01/`: 第1巻用
- `volume_02/`: 第2巻用
- `volume_03/`: 第3巻用
- `spinoff/`: スピンオフ用

## おすすめの使い方
1. まず `00_planning/trilogy_plan.md` を開き、第1巻の章構成や仮タイトルを埋めてみてください。
2. 実際に書き始めたら、進捗状況をこのファイルで更新するとモチベーション維持に役立ちます。
